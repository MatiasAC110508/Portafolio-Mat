# Portafolio Personal

Este repositorio contiene mi portafolio personal, donde presento información sobre mí, mis proyectos y una sección dedicada a mis mascotas. El objetivo del proyecto es practicar y demostrar conocimientos básicos e intermedios de HTML y CSS, aplicando diseño responsive y buenas prácticas de maquetación web.

---

## Contenido del proyecto

El portafolio está compuesto por las siguientes páginas y secciones:

* Página principal (index.html)
  Presentación personal, descripción, imagen destacada y estructura general del portafolio.

* Página de mascotas (mascotas.html)
  Página dedicada a mi mascota Walter, con imágenes y una breve historia.

* Header y navegación
  Barra de navegación con título y enlaces entre páginas.

* Main
  Contiene las secciones principales de cada página.

* Footer
  Información de contacto y enlaces externos.

---

## Tecnologías utilizadas

* HTML5: estructura del contenido.
* CSS3: estilos, diseño visual y responsive.
* Flexbox: distribución y alineación de elementos.
* Media Queries: adaptación a diferentes tamaños de pantalla.

---

## Diseño y estilos

El diseño del portafolio se basa en:

* Fondo con imagen y degradado oscuro para mejorar la legibilidad.
* Uso de transparencias y backdrop-filter para un efecto visual tipo glassmorphism.
* Paleta de colores oscuros con acentos en tonos morados y azules.
* Animaciones suaves con transition y efectos hover.

El layout principal utiliza Flexbox en columna para asegurar que el footer permanezca en la parte inferior de la página, incluso cuando el contenido es corto.

---

## Diseño responsive

El portafolio está adaptado para dispositivos móviles mediante:

* Uso de porcentajes y max-width.
* Media query principal en max-width: 900px.
* Reorganización del menú de navegación en pantallas pequeñas.
* Ajuste de imágenes y contenido para mejorar la experiencia en móvil.

---

## Imágenes

* Las imágenes se manejan con object-fit: cover para evitar deformaciones.
* Se utilizan tamaños controlados para evitar pixelación.
* Las imágenes cuentan con bordes redondeados, sombras y efectos al pasar el cursor.
